Question 1a

The NAND gate, which stands for NOT-AND is quite intuitive. The output of its truth table is described by the inverse of a regular AND gate.
As such, a NAND gate is simply built from an AND gate followed by a NOT gate.


Question 1b

The NAND gate is functionally complete, any algebraic or truth table can thus be implemented using only NAND gates.
Below you can see the AND, OR and NOT gates implemented using only NAND gates.
